# SMALTO Nail Bar

Sitio web estatico para marketing y captacion de citas de SMALTO Nail Bar.

## Despliegue en Vercel

1. Sube el contenido de esta carpeta completa a un repositorio de GitHub.
2. En Vercel, importa ese repositorio.
3. Deja el directorio raiz del proyecto como la carpeta donde estan `index.html`, `styles.css` y `assets/`.
4. No cambies ni omitas la carpeta `assets/`; ahi viven todas las fotografias del sitio.
5. El sitio incluye una agenda visual en `script.js`. Para confirmar citas reales, conecta el formulario a WhatsApp Business, Calendly, Google Calendar o tu sistema de reservas.

## Estructura

- `index.html`: contenido de la landing page.
- `styles.css`: diseno responsivo.
- `script.js`: agenda interactiva con servicios, precios, duracion, nail techs y horarios disponibles.
- `assets/`: fotografias del salon y trabajos de unas.
- `vercel.json`: configuracion minima para Vercel.
