# SMALTO Nail Bar

Sitio web estatico para marketing y captacion de citas de SMALTO Nail Bar.

## Despliegue en Vercel

1. Sube esta carpeta a un repositorio de GitHub.
2. En Vercel, crea un nuevo proyecto e importa el repositorio.
3. Usa la configuracion por defecto para un sitio estatico.
4. Si tienes un enlace directo de WhatsApp, Calendly o reservas, reemplaza las URLs de Google Maps en `index.html`.

## Estructura

- `index.html`: contenido de la landing page.
- `styles.css`: diseno responsivo.
- `assets/`: fotografias del salon y trabajos de unas.
